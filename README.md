# FINANCE Dashboard
Interactive Finance Summary Dashboard built in Excel to track business performance. Visualizes key KPIs like Revenue, Expenses, and P&L alongside categorical and departmental deep dives. Features monthly profit trends and multi-criteria interactive slicers (Year, Quarter, Department) for seamless, data-driven financial analysis.
